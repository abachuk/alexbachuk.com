jQuery Mobile Active Page URL

Feb 13, 2013 #jquery mobile #mobile web development #quick tip

jQuery mobile contains several pages in DOM at once to make page transition smooth and fast, and also pre-caching static pages. If you need to get current URL, and decided to use

window.location.href

  • it will return previous page URL, because active page is inserted into the DOM via AJAX. To get current page URL use

    $.mobile.activePage[0].baseURI
