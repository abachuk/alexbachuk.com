AJAX is a big part of JavaScript and one of the reasons this language is so popular. It’s possible to get any content from the server and send any content back to server, two way communication. Libraries and frameworks like Backbone.js make this process very elegant and organized. Few years ago it was a standard proces to append html in AJAX callback to the DOM, and it works, and if it’s done right (append all the content at once instead of at each iteration of a loop), it’s even ok. »
